# TDM Classification and Masking JSON Viewer


**Beta Tool Notice**

This is a free to use tool built by Solution Engineering to assist with understanding the JSON outputs from the CLI. This viewer is provided as a purely Beta tool for viewing and exploring TDM classification and masking JSON files. No support or warranty is provided for this tool. Use at your own risk.

## Features
- View and filter classification and masking JSON files
- Session persistence for uploaded files
- Advanced filtering switches for quick table views

## Usage
1. Upload your own `classification.json` and `masking.json` files using the JSON Upload tab.
2. Use the Classification and Masking tabs to explore and filter your data.

> **Note:** The JSON files provided in this repository are dummy test files for illustration purposes only. Please use your own files for real data and analysis.
